# CSV Demo Project

This is a simple Python project to demonstrate:
- Writing CSV files
- Reading CSV files
- Using Python with GitHub

## How to Run
1. Clone this repository:
git clone https://github.com/ngs0707/csv_demo_project.git
2. Navigate into the folder:
cd csv_demo_project
3. Run the Python script:
python csv_demo.py

The script will create `countries.csv` in the folder and display its contents.
